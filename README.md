# ✈️ Quokka Travel

QuokkaTravel은 쿼카와 함께 여행을 다니며 여행지의 사람들과 커뮤니케이션을 할 수 있는 서비스입니다.

> 여행 숙소 정보를 확인하고, 여행이 시작되면 여행지의 사람들과 **대화**를 나눌 수 있습니다.
> 여행지의 다양한 이벤트와 함께 행복한 여행을 할 수 있습니다.

### 기능 1

- 여행지 숙소 예약

### 기능 2

- 여행지에 대해서 다양한 사람들과의 대화를 할 수 있는 오픈채팅
